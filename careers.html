Careers

Content Team

All the selected candidates are general content creators. A few selected are given the responsibility of:

Content Router

  • Assign incoming content work from IKS MinT and IEEE Kerala Section among members.
  • Make sure work is distributed equally and is done properly.
  • Distribute content work to members for IKS Inked and IKS Interns Events.
  • Coordinate with the team and help with timely posting of content along with release of posters.
  • Acknowledging MinT related Emails and requests.

Content Curator - Inked

  • Create and improvise created content for IKS Inked.
  • Filter out and select articles to be published in IKS Inked.
  • Coordinate with the team in timely release of the magazine.

Design Team

Associate Designer - UI

  • Improve the look and feel of IKS MinT and other IEEE Kerala Section Products & websites.
  • Create overall concepts for the user experience within MinT.
  • Design the aesthetics to be implemented within IKS MinT and other IEEE Kerala Section Products & websites.
  • Create surveys for research through various social media platforms to gather feedback in understanding seamless application experience for users.
  • Conduct tests to assess user experience.
  • Coordinate with the web team to implement the UI and make iterations.
  • Should have knowledge on any of the UI design tools.

Associate Designer - Inked

  • Design layout for IKS Inked Magazine.
  • Create predefined content arrangement layouts for the magazine.
  • Suggest new design iterations to make the magazine more attractive.
  • Coordinate with the content team in timely release of the magazine.

Associate Designer - External

  • Design posters for IEEE Kerala Section and IKS Interns events.
  • Create posters for important days like World Environment day, National Technology day, etc.
  • Design posters for requirements from IKS MinT.
  • Create predefined poster templates for events.
  • Coordinate with the team and help with timely release of posters.
  • Should have experience in conducting events or sessions.

Associate Video Editor

  • Create promotional videos for IEEE Kerala Section and IKS Interns events.
  • Create videos for requirements from IKS MinT.
  • Create promotional videos for new versions of IKS MinT.
  • Create tutorial videos for IKS MinT.
  • Coordinate with the team and help with timely release of videos.
  • Should have thorough knowledge in necessary Adobe Suites.
  • Should be ready to make necessary changes in a given time interval.

Web Team

Backend Developer

  • Should have a good knowledge of Node.js, SQL and JavaScript.
  • Manage existing backend API and services in IKS MinT.
  • Work on the implementation of new services in IKS MinT.
  • Connect backend with 3rd party APIs for additional functionality in IKS MinT.
  • Work on improving the performance of the backend and work on code optimisation in IKS MinT.
  • Coordinate with the team to implement new versions of IKS MinT.

Frontend Developer

  • Should have a good knowledge of React or any other front end JS frameworks, State Management techniques and JavaScript.
  • Work on the implementation of new services in IKS MinT.
  • Work on new aesthetic and user friendly designs along with efficient data management for IKS MinT.
  • Work on improving the performance of the frontend and work on code optimisation in IKS MinT.
  • Coordinate with the team to implement new versions of IKS MinT.

DevOps

  • Should have good knowledge of cpanel and devOps tools.
  • Manage and coordinate between deployment of different versions of frontend and backend in IKS MinT.
  • Manage the support mails of IKS MinT and resolve issues by communicating the issues with the corresponding teams.
  • Coordinate with the team and execute activities in IKS MinT.

WebOps

  • Should have good knowledge of PHP, cpanel and DevOps Tools.
  • Manage IEEE Kerala Section website and other web assets.
  • Write code to optimize and manage the existing web assets and also for new requests from IKS MinT.
  • Coordinate with the team and create forms and other required web resources for events.
